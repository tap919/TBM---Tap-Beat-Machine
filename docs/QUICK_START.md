# TBM Quick Start Guide

## Create Your First Beat in 5 Minutes

### Step 1: Launch TBM
1. Open TBM in your browser or desktop app
2. You'll see the main interface with 16 pads, a sequencer, and mixer

### Step 2: Load Drum Samples
1. **Method A (Built-in samples):**
   - Click any pad (e.g., pad 1)
   - Click the folder icon that appears
   - Select "Kick" from the sample browser
   - Repeat for pads 2-4 with "Snare", "Hi-Hat", "Clap"

2. **Method B (Your own samples):**
   - Find drum samples on your computer
   - Drag and drop them onto pads 1-4
   - Recommended: kick, snare, hi-hat, percussion

### Step 3: Program a Basic Pattern

#### Kick Drum (Pad 1)
1. Select pad 1 (kick drum)
2. In the step sequencer (top-right), click steps:
   - **1, 5, 9, 13** (every 4th step)
   - This creates a four-on-the-floor pattern

#### Snare (Pad 2)
1. Select pad 2 (snare)
2. Click steps:
   - **5, 13** (on beats 2 and 4)

#### Hi-Hat (Pad 3)
1. Select pad 3 (hi-hat)
2. Click steps:
   - **1, 3, 5, 7, 9, 11, 13, 15** (every other step)

### Step 4: Play Your Pattern
1. Click the **Play** button (triangle) or press **Spacebar**
2. You should hear your drum pattern looping
3. Adjust the **BPM** slider to change tempo (try 120-140 BPM)

### Step 5: Add Variation

#### Add Swing
1. Find the **Swing** knob (near BPM)
2. Turn it to **55-60%**
3. Listen to how the groove changes

#### Humanize the Hi-Hats
1. Find the **Humanize** knob
2. Turn it to **20-30%**
3. This adds natural timing variations

### Step 6: Record a Bassline

1. Switch to the **Piano Roll** tab (center area)
2. Click the **+ Track** button to add a bass track
3. Select a bass sound for the track
4. Click **Record** (red circle) or press **Enter**
5. Play some bass notes on your MIDI keyboard or computer keyboard:
   - **A** = C, **W** = C#, **S** = D, **E** = D#, **D** = E, **F** = F
   - Try a simple pattern: C, E, G (hold each for 1-2 beats)

### Step 7: Add Effects

#### Reverb on Snare
1. Find the **Effects Rack** (right side)
2. Drag a **Reverb** effect onto the snare channel
3. Adjust:
   - **Size**: Medium
   - **Decay**: 1.5 seconds
   - **Mix**: 25%

#### Filter Sweep
1. Add a **Filter** effect to the hi-hat channel
2. Set to **Low-pass**
3. Automate the cutoff:
   - Click the automation lane (below sequencer)
   - Select "Filter Cutoff"
   - Draw a rising curve over 16 steps

### Step 8: Mix Your Track

#### Balance Levels
1. Look at the **Mixer Console** (bottom)
2. Adjust faders:
   - **Kick**: 0 dB (reference level)
   - **Snare**: -3 dB
   - **Hi-Hats**: -6 dB
   - **Bass**: -3 dB

#### Add Sidechain (Optional)
1. Enable **Sidechain** on the bass channel
2. Set source to **Kick**
3. Adjust amount until bass "ducks" slightly with each kick

### Step 9: Export Your Beat

1. Click **File** → **Export Audio**
2. Choose format: **WAV** (for quality) or **MP3** (for sharing)
3. Select length: **Pattern** (current loop) or **Entire Project**
4. Click **Export** and choose save location

## Next Steps

### Explore More Features
1. **Sample Chopping**: Load a vocal sample and trigger different sections
2. **Stem Separation**: Upload a song and extract drums/bass/vocals
3. **Modulation Matrix**: Create evolving sounds with LFOs
4. **VST Plugins**: Load your favorite effects (desktop app only)

### Save Your Project
1. **File** → **Save Project As**
2. Name your project
3. All samples, patterns, and settings are saved

### Practice Exercises
1. **Exercise 1**: Create a hip-hop beat at 90 BPM
2. **Exercise 2**: Make a house track with sidechain pumping
3. **Exercise 3**: Chop and rearrange a vocal sample
4. **Exercise 4**: Create a complete song structure (intro, verse, chorus)

## Tips for Success

### Workflow Tips
1. **Start simple** - Get basic elements working first
2. **Save often** - Use Ctrl+S every few minutes
3. **Use templates** - Save your favorite setups
4. **Experiment** - Try unexpected combinations

### Common Pitfalls
1. **Too many sounds** - Start with 4-5 elements
2. **Over-complicated patterns** - Simple often sounds better
3. **Ignoring mixing** - Balance levels from the start
4. **Not saving** - Don't lose your work!

## Need Help?

- Check the **User Guide** for detailed instructions
- Look for **tooltips** (hover over controls)
- Check **console** for error messages
- Reduce complexity if experiencing performance issues
